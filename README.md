# Sistema básico de usuarios.

Proyecto creado con Flask que permite:
- Registrar usuarios en un Html básico.
- No guarda usuarios con nombres duplicados.
- Tiene persistencia de datos mediante JSON.

## Tecnologías:
- Python.
- Flask.
- HTML.

### Cómo ejecutarlo:
1. Instalar dependencias:
  - pip install flask
2. Ejecutar:
  - python app.py
